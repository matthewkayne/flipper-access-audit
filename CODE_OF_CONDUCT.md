# Code of Conduct

## Expected behaviour

- Be respectful and constructive in all discussions
- Keep discussion relevant to the project and its authorized use cases
- Welcome contributors of all experience levels
- Accept feedback gracefully

## Unacceptable behaviour

- Harassment, personal attacks, or discriminatory language
- Discussing or assisting with unauthorized use of this tool
- Sharing results from systems you do not own or have permission to test
- Spam or off-topic promotion

## Enforcement

Issues or pull requests that violate this code of conduct will be closed. Repeated violations will result in being blocked from the repository.

To report a conduct issue, contact the maintainer directly via GitHub.
