# Flipper Access Audit

A Flipper Zero app for defensive auditing of access-control credentials.

## Status

Early prototype. The app builds with uFBT and is currently being developed toward a first MVP.

## Goal

Classify a credential, apply simple audit heuristics, and show a human-readable risk assessment for authorized testing.

## Planned MVP

- Detect credential family
- Apply simple audit rules
- Produce a risk score
- Show findings on-device
